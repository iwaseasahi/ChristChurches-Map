***REMOVED******REMOVED*** 概要
***REMOVED******REMOVED******REMOVED*** ・キリスト教会マップは、日本全国にあるプロテスタント教会をマップに表示します。
***REMOVED******REMOVED******REMOVED*** ・教団教派・グループ・都道府県・名前・住所で検索することができます。
***REMOVED******REMOVED******REMOVED*** ・教会に行ったことがない方でも、その教会がどのような雰囲気か知ることができるようにします。
